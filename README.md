# Hexadevs

### Aprende desarrollo construyendo proyectos reales.

Hexadevs es una academia online enfocada en desarrollo de software moderno y aprendizaje práctico.

Creamos cursos escritos, proyectos guiados y contenido pensado para enseñar herramientas, arquitecturas y flujos de trabajo reales.

La idea detrás de Hexadevs es simple:

> aprender programación creando cosas reales, no solo leyendo teoría.
> 
---

Actualmente estamos construyendo:

* 📚 Cursos escritos
* 🛠 Proyectos guiados
* 🧠 Quizzes y exámenes
* 🏆 Logros y rachas
* 💬 Comentarios y comunidad
* 📰 Blog de desarrollo
